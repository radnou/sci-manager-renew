# Documentation Produit

Ce dossier centralise la documentation business et fonctionnelle du projet.

## Fichiers

- `BUSINESS_FUNCTIONAL_OVERVIEW.md`: vision, proposition de valeur, cibles et priorités business.
- `FUNCTIONAL_REQUIREMENTS.md`: périmètre fonctionnel, parcours clés, user stories et backlog MVP.
- `GTM_AND_METRICS.md`: stratégie go-to-market, funnel et KPI de pilotage.
- `VPS_PREPARATION_RUNBOOK.md`: runbook opérationnel pour préparer un VPS et exécuter un déploiement distant SSH.

## Usage recommandé

1. Lire `BUSINESS_FUNCTIONAL_OVERVIEW.md` pour cadrer la stratégie.
2. Traduire en exigences concrètes via `FUNCTIONAL_REQUIREMENTS.md`.
3. Piloter l'exécution commerciale avec `GTM_AND_METRICS.md`.
4. Préparer l'infrastructure avec `VPS_PREPARATION_RUNBOOK.md` avant intervention production.
